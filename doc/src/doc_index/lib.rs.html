//! This is the index page for the documentation to the package [`rly`]. The package
//! consists of three crates:
//!
//! * [`lex`] for creating [Lexers] capable of parsing an input into [`Token`'s].
//!
//! * [`parse`] for creating [Parsers] that can parse a stream of [`Tokens`] into an
//! [`Ast`].
//!
//! * [`rly_common`] containing code common to both [`lex`] and [`parse`]. Most
//! significant for the end-user are the types [`ErrorData`] and [`LineOffset`].
//!
//! [`rly`]: https://github.com/dsmcdermott/rly
//! [Lexers]: lex#lexers
//! [`Token`'s]: lex::Token
//! [Parsers]: parse#using-parsers
//! [`Tokens`]: lex::Tokens
//! [`Ast`]: parse::ast::Ast
//! [`ErrorData`]: rly_common::errors::ErrorData
//! [`LineOffset`]: rly_common::errors::LineOffset
